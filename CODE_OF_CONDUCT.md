# Code of Conduct

## Our pledge

We want Sleep Time to be a welcoming, respectful open source project.

## Expected behavior

- be respectful
- assume good intent
- give constructive feedback
- focus on the code and product, not the person

## Unacceptable behavior

- harassment
- hate speech
- threats
- doxxing
- repeated bad-faith disruption

## Enforcement

Project maintainers may remove comments, reject contributions, or ban participants who violate this code of conduct.
